```

  ├── 签名: HS256 with JWT_SECRET
```

## 📞 问题排查快速指南

| 问题 | 原因 | 解决方案 |
|-----|------|--------|
| "Cannot find module" | 缺少依赖 | `npm install` |
| CORS错误 | 前端URL不匹配 | 检查 .env 中 FRONTEND_URL |
| 401 Unauthorized | 没有发送token | 检查axios拦截器是否添加 Authorization 头 |
| 数据库连接失败 | 凭证错误 | 验证 SUPABASE_URL 和 KEY |
| 端口3001已使用 | 另一个进程占用 | `lsof -i :3001` 并 kill 进程 |

**详细排查**: 见 INTEGRATION_GUIDE.md 的"故障排除"部分
