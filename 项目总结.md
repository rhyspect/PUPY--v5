# 🎯 PUPY 项目总结

## 📱 关于PUPY

PUPY是一个创新的宠物社交平台，将现实宠物护理与云端AI伴侣结合，为宠物主人和宠物们创造互联互通的社交体验。

### 核心特性

✨ **双重身份系统**
- 真实宠物档案 + AI克隆
- 实时位置匹配
- 性格MBTI分类

🗺️ **奇幻领域**
- 躲雨深林（社恐宠物安全港）
- 气泡失重海（跨地域社交）
- 霓虹天台公园（同城约遛）

💬 **智能匹配**
- 卡片滑动式发现
- 相容性算法
- 实时消息系统

🛍️ **宠物市集**
- 宠物超市
- 溜狗服务
- 宠物恋爱配对
- 专业护理服务

## 💾 数据库架构

### 9个核心表

| 表名 | 用途 | 关键字段 |
|------|------|--------|
| users | 用户管理 | id, email, name, mbti, location |
| pets | 宠物档案 | id, user_id, type, mbti, is_real |
| matches | 匹配记录 | pet_a_id, pet_b_id, status, score |
| messages | 消息系统 | sender_id, recipient_id, match_id |
| realms | 社交空间 | name, personality_type, online_count |
| user_stats | 用户统计 | user_id, level, exp, badges |
| market_items | 市场商品 | user_id, category, price, rating |
| notifications | 通知系统 | user_id, type, is_read |
| reports | 安全举报 | reporter_id, reported_user_id |

## 🚀 快速开始（5分钟）

### 后端

```bash
cd "c:\Users\selin\Desktop\PUPY-爪住\后端V2"

# 1. 安装依赖
npm install

# 2. 配置Supabase
# 编辑.env，填入你的Supabase凭证

# 3. 数据库迁移
npm run db:migrate

# 4. 启动服务
npm run dev
```

### 前端

```bash
cd "c:\Users\selin\Desktop\PUPY-爪住\前端V2"

# 1. 安装依赖
npm install

# 2. 配置API URL
echo "VITE_API_URL=http://localhost:3001/api" > .env.local

# 3. 启动开发
npm run dev
```

### 访问应用

- 前端: http://localhost:5173
- 后端: http://localhost:3001
- API文档: http://localhost:3001/api/health

## 🛠️ 技术栈

- **运行时**: Node.js 18+
- **语言**: TypeScript
- **框架**: Express.js
- **数据库**: Supabase (PostgreSQL)
- **认证**: JWT + Bcrypt
- **包管理**: npm

- **语言**: TypeScript + React
